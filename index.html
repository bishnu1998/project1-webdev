HTML Summary

HTML(HyperText Markup Language) is used to define our page content,structure and meaning. you don't use it for styling purposes. Use CSS for that instead!

  • HTML uses 'elements' to describe(annotate) content
  • HTML elements typincally have an openig tag, content and then closing tag
  • You can also have void(empty)elemetnts like images
  • You can also configure elements with attributes
  • There's a long list of elements but you'll gain experience over time. no worries Learn more about time,no worries.

css Summary

CSS(Cascading style sheets) is used for styling your page content.

  • Styles are assigned via proparty value pairs
  • You can assign styles via the 'style' attribute
  • To avoid code duplication,you typicall use global styles(e.g via the 'style' element)insted
  • Alternatively, you can work with exeternal stylesheet files which you 'link' to
  • When working with CSS, concepts like 'inherittance','specificity'and the 'box model' should be understood
